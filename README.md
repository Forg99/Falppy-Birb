# Falppy-Birb

Flappy bird is a mobile game developed in 2013 about a bird flying between pipes. the game i made is badly made clone of it.

Controls: when pressing spacebar the bird flies up, if the bird is not going upwards it will go downwards.

gameplay: pipes will spawn in pairs with a gap to right of the screen, hitting a pipe or the edges of the screen results in death.

points: 1 point is scored each time the bird flies in between a pair of pipes.

Menu: upon launching the game a menu will appear with a start button. after dying a button will appear to play again.
